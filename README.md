# 🧹 Stripper

> **Ultimate Windows ISO Debloater** — PowerShell interactive script using wimlib, NTLite preset XML. Mount ISO, apply debloat, rebuild.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📂 **File Picker** | Select ISO and NTLite XML preset |
| ⚙️ **Switches** | -KeepStore, -KeepXbox, -KeepDefender, -KeepEdge, -KeepUpdates |
| 📦 **wimlib** | Downloads wimlib-imagex for WIM operations |
| 📋 **NTLite Preset** | Applies XML debloat preset to image |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11 |
| **PowerShell** | 5.1+ |
| **Privileges** | Administrator |

---

## 🚀 Usage

```powershell
.\Stripper.ps1
.\Stripper.ps1 -KeepStore -KeepDefender
```
